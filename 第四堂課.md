### 專案執行步驟
1.使用VS Code打開檔案夾，確認是打開src的上一層檔案夾。  
2.打開終端機  
3.執行以下指令，成功會產生node_modules檔案夾  
```
npm install
```
4.執行以下指令，完成後可開啟網頁。  
```
npm run dev
```
5.之後只需執行 `npm run dev` 即可跑起專案。
